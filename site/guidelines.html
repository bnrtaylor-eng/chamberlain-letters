Editorial Guidelines

Contributors to the Chamberlain Letters Project are asked to follow these editorial practices to ensure consistency, clarity, and scholarly rigor:

  • Use the TEI template provided for encoding each letter.
  • Always close XML tags. For example: <persName>John Chamberlain</persName>
  • Tag entities using TEI-standard elements: <persName>, <placeName>, <orgName>, <title>.
  • Use the ref attribute to link to authoritative external sources (Wikipedia, VIAF, ODNB, etc.).
  • Add an <editor-note> summarizing the letter for general readers, including historical context.
  • Add in-line <note place="foot"> elements to capture footnotes or explanatory comments.
  • Leave unresolved IDs for people/places/works as ref="#unknown" and include a <note type="editorial"> suggesting possible identities.
  • Send completed XML files to the project editor for verification before publication.
